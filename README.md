# whiteBoard

## To Run Locally 
### 1. clone repo
### 2. run ```npm install``` in /whiteboardBackend and /konvajs directories
### 3. run ```docker compose up``` in root porject directory
### 4. run ```npm run dev``` in /whiteboardBackend and /konvajs directories
